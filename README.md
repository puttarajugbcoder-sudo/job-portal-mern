# Job Portal MERN

A full-stack Job Portal application built using the MERN Stack.

## Tech Stack

### Frontend

* React.js
* Vite
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## Features

* User Authentication
* Job Listings
* Apply for Jobs
* Recruiter Dashboard
* Resume Upload
* Role-Based Access
